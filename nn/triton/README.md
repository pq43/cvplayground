# triton Implementation of BabyCNN

## Overview
This folder contains the implementation of BabyCNN using triton.

## Setup Instructions
### Dependencies
- triton

### Setup
```sh
pip install triton
```

## Training
Run the following command to train the model:
```sh
python train.py
```

## Inference
Run the following command to perform inference:
```sh
python infer.py
```
